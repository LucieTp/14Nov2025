(1) Installer pycharm, un IDE python (ça prend 20 minutes, mais c'est le plus simple pour que tout fonctionne sans pb)
https://www.jetbrains.com/pycharm/data-science/?var=anaconda 

(2) Télécharger tout le contenu de ce repo github (le bouton vert "code" > "download zip" tout en bas du menu déroulant

(3) Installer les modules pygame etc. Pour cela, dans pycharm, vas dans la section "terminal" (dans les boutons du menu à gauche) et execute pip install pycharm, 

(4) Ouvrir le projet Pycharm (dans pycharm - ouvrir > projet existant > localisation de ton fichier dézippé)

(4) normalement tout est prêt! Ouvre le script "main.py" et click sur la flèche verte (run)
